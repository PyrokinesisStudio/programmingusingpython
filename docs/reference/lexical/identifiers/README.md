> Lexixal definition of an ```identifier``` (a.k.a. ```name```)  Python

* Unlimited in length
* Case-sensitive

# Python 2.x

* https://docs.python.org/2.7/reference/lexical_analysis.html#identifiers

```python
identifier ::=  (letter|"_") (letter | digit | "_")*
letter     ::=  lowercase | uppercase
lowercase  ::=  "a"..."z"
uppercase  ::=  "A"..."Z"
digit      ::=  "0"..."9"
```
